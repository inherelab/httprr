# HttpRR

A simple HTTP Request & Response Service.

> refer from http://httpbin.org

## Install

```bash
git clone https://github.com/inherelab/httprr.git
```

## Start

```bash
go run main.go
```

## Docker

```bash
docker run -p 8080:8080 inherelab/httprr
```
